README.md

پروژه آزمایشی COW

این پروژه یک نمونه آموزشی از زبان برنامه‌نویسی COW است.

COW یک زبان برنامه‌نویسی تفننی (Esoteric Programming Language) است که تقریباً تمام دستورات آن از کلمه moo و ترکیب‌های مختلف حروف بزرگ و کوچک تشکیل شده‌اند.

هدف این پروژه آشنایی با ساختار برنامه‌های COW است.

ویژگی‌ها

- نمونه برنامه COW
- مستندسازی فارسی
- مناسب برای یادگیری اولیه
